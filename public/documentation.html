Movie API

The purpose of this project is to build the client-side component of this application using REACT. The next project will be to have a full-stack web app using MERN (MongoDB, Express, React, and Node.js) stack. With this project I hope to learn and demonstrate proficiency in full-stack JavaScript development, including APIs, web server frameworks, databases, authentication, and more.

Essential Features:

  • Return a list of ALL movies
  • Retrieve data about a single movie by title
  • Retrieve data about a genre by name/title
  • Retrieve data about a director by name
  • Allow new users to register
  • Allow users to update their info
  • Allow users to add/remove movies to/from favorites
  • Allow existing users to deregister
